Our Process

Here is an overview of how our application works.

Plan

  • Client interview
  • Gather fitness data
  • Create strength/condition strategy
  • Analyze research

Create

  • Create user account
  • Build exercercises
  • Create workouts
  • Assign workouts to client

Assess

  • Assessment of client
  • Modify workouts around assessment scores
  • Track client progress
  • Review gathered data with client
